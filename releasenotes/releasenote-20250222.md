# Terrakko version 1.0 Release Notes

## Introduction to Terrakko v1.0

The latest Terrakko release, version 1.0, is finally available!  
Terrakko will not make major updates as long as Nekko Cloud uses Proxmox VE.  

## Changes to the Discord bot

To prevent destructive changes to the production environment — a concern not addressed in the alpha version — VM instances can now be created, modified, and deleted based on Proxmox VE access rights.  
In addition, fixes have been made across all programs to improve readability and maintainability.
