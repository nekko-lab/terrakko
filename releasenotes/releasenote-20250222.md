# Terrakko version 1.0 Release Notes

## Introduction to Terrakko v1.0

The latest Terrakko release, version 1.0, is finally available!  
Terrakko will not make major updates as long as Nekko Cloud uses Proxmox VE.  

## Changes to the Discord bot

As a measure to prevent destructive changes to the production environment, which was not taken into account in the alpha version, VM instances can now be created, modified, and deleted based on Proxmox VE access rights.  
In addition, Fixes have been made to improve readability of all programs to maintain source code maintainability.
